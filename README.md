## Quick start

Copy the project code from the repository.

```powershell
git clone https://github.com/ManucherKM/vacation.git
```

Go to the project directory.

```powershell
cd vacation
```

Install dependencies.

```powershell
npm install
```

Run the project in development mode.

```powershell
npm run dev
```
